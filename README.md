# Testes de Hipótese com Python

Este projeto contém um notebook Jupyter voltado ao estudo de **testes de hipótese estatísticos**, com foco em conceitos fundamentais, diferentes tipos de testes e aplicações práticas em Python.

## Conteúdos
- Hipótese nula e alternativa  
- Testes paramétricos e não paramétricos  
- Testes de normalidade  
- Correlação e análise estatística  
- Visualização de resultados  

## Tecnologias
- Python  
- Jupyter Notebook  
- pandas, numpy  
- scipy  
- plotly  

## Execução
Instale as dependências e execute o notebook:

```bash
pip install -r requirements.txt
jupyter notebook
```

## Objetivo
Material educacional para estudo de estatística inferencial aplicada à análise de dados.
